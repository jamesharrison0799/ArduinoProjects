User input will be done via the Serial Monitor, in later versions a keypad will be used.

Tasks:

 - Take decimal number from user (8 digit number (Eg: ID number)), RGB LED must be GREEN
 - convert to binary, display binary numbers in sequence, RBG LED must display RED
 - Take encryption key from user (4 digit pin code)
 - Encrypt 8 digit number
 - Display number (In a loop), RGB LED must display RED
